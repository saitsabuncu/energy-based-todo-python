# Energy Based Todo List (Python)

Beginner-friendly Python CLI project.

## Features
- Add tasks with energy levels
- Show tasks based on current energy
- Mark tasks as completed
- Daily energy summary

## Why this project?
Because productivity is not about time, it's about energy.
